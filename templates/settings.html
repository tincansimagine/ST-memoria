질문
작동 방식 — AI 응답이 끝날 때마다 그 턴을 자동으로 기록해 구조화된 기억으로 보관하고, 매 턴 현재 대화와 관련 있는 기억만 골라 토큰 예산 안에서 주입합니다. 스와이프·편집·삭제 시 해당 턴의 기억은 자동으로 무효화됩니다.

기억 브라우저

고정(📌)한 기억은 매 턴 반드시 주입됩니다.

기억 추가

인물 도감

대화에 등장한 인물의 프로필과 관계입니다. 새 인물이 등장하면 자동으로 기록되고, 매 턴 AI에게 전달됩니다.

인물 추가

아이템 도감

스토리에 의미 있는 물건들. 소유자와 상태가 자동으로 갱신됩니다.

아이템 추가

서약

비밀·미해결 떡밥·건너뛰면 안 되는 전개 등, 이야기가 지켜야 할 약속. 토큰이 부족해도 가장 마지막까지 유지됩니다.

서약 추가

캐논 (세계 설정)

세계관의 지속적 사실. 같은 키가 다시 기록되면 덮어씁니다.

상태 보드

인물·사물·장면의 현재 값(위치, 복장, 부상, 목표…). 슬롯별 최신 값만 유지됩니다.

자료실

원작 소설·설정 문서를 붙여넣어 두면, 매 턴 지금 장면과 관련된 대목만 골라 배경 참고로 주입합니다. 채팅에서 확정된 사실이 항상 우선하며, 토큰 예산이 부족하면 가장 먼저 줄어듭니다.

문서 추가

지금까지의 이야기

N턴마다 자동 요약되고, 오래된 요약은 연대기로 병합됩니다. {{memoria_story}} 매크로로도 쓸 수 있습니다.

지금 요약

턴별 기록

사서가 남긴 턴 단위 기록입니다. 요약은 이 기록을 엮어 만들어지므로, 여기서 고치면 다음 요약부터 반영됩니다.

주요 사건 연표

이야기의 방향을 바꾼 결정적 순간들을 시간순으로 정리한 연표입니다.

기본

추적 도감

켜면 사서가 대화에서 자동으로 골라내 도감에 기록하고, 매 턴 주입에 포함합니다.

모델 (사서 API)

기억 기록·요약을 담당하는 사서가 쓸 API입니다. 저렴하고 빠른 모델을 권장합니다.

연결 테스트

의미 검색 임베딩 (선택)

임베딩을 켜면 회상 정확도가 크게 올라갑니다. 동의어·의역도 잡아냅니다. 실패 시 자동으로 내장 검색으로 폴백합니다.

실리태번 서버가 임베딩 모델을 직접 돌립니다. 무료·무제한이며 API 키가 필요 없습니다. 처음 사용할 때 모델(수십 MB)을 한 번 내려받느라 잠시 걸릴 수 있습니다.
